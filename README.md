# jeshop
  jeshop是一个 SSM + Maven 的购物商城网站，使用瀑布模型进行开发。主要功能有用户管理、订单管理、购物车管理以及商品展示四个部分。
该项目采用MVC的基本模式，使用SSM框架进行程序的开发，Maven进行依赖管理和项目构建，数据库采用Mysql。

### 软件环境：Windows

### 开发工具：IDEA

### 项目部署: 直接下载到编译器上，在Eclipse或者IDEA中修改配置文件即可。
